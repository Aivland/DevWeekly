PyHubWeekly是一个精选Github上优质Python小工具的项目，宗旨，

- 每周更新一次
- 精选GitHub上优质Python项目

本项目不追求数量而追求质量，不会为了拼凑数量而一味的去美化一个项目，把它描绘的天花乱坠。
